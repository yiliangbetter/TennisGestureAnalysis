# Instructions for connecting to GitHub

Once you've created your repository on GitHub, run these commands:

1. Add the remote origin (replace with YOUR repository URL):
   git remote add origin https://github.com/YOUR_USERNAME/TennisGestureAnalysis.git

2. Verify the remote is set correctly:
   git remote -v

3. Push your code to GitHub:
   git push -u origin main

Your repository is now ready to be pushed to GitHub!
